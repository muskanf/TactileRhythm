What it generates

  • Table representation Rows for instruments and columns for rhythm positions.
  • Navigable representation A position-by-position text explanation for screen reader users.
  • Tactile SVG preview A grid-based tactile-style graphic with markers for played notes.

Syntax guide

title:
Names the rhythm.
time:
Stores the time signature, such as 4/4.
x
The instrument plays at this position.
.
The instrument rests at this position.
